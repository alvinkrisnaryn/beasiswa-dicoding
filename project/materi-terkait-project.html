Membuat web pertama

  1. Windows memiliki code editor bawaan yang bernama Notepad. Kita akan menggunakan aplikasi tersebut untuk membuat halaman web. Ada banyak cara yang dapat dilakukan. Salah satunya adalah melalui fitur run command atau pencarian aplikasi - pencarian aplikasi yang dimaksud adalah pop up yang muncul jika menekan tombol Windows dan terdapat pencarian aplikasi padanya. Silakan buka pencarian aplikasi tersebut dengan menekan tombol Windows. Jika sudah, tuliskan “notepad” dan pilihan aplikasi Notepad akan muncul. Kliklah pilihan. aplikasi tersebut untuk membuka Notepad.
  2. Selanjutnya, kita akan mulai menuliskan konten untuk halaman web kita. Sebetulnya, ada struktur yang disepakati untuk menuliskan halaman web. Saat ini masih dalam tahap awal membuat halaman web, jadi kita akan menuliskan kontennya secara langsung. Silakan isi dengan konten berikut pada Notepad.
  3. Nah, kita sudah selesai menuliskan konten halaman web. Sekarang, kita dapat melihat hasilnya pada browser. Namun, kita perlu menyimpan perubahan isi berkas terlebih dahulu. Silakan lakukan simpan berkas. Anda dapat melakukannya dengan kombinasi tombol ”CTRL + S”. Kemudian, pilih lokasi penyimpanan yang diinginkan. Contoh pada komputer kami, kami menyimpannya pada alamat “C -> WebDasar”. Oleh karena itu, buka direktori “WebDasar” dengan dobel klik kiri pada folder terkait.
  4. Seharusnya, tampilannya sudah berada dalam direktori WebDasar saat ini. Selanjutnya, kita namai berkas ini dengan “index.html” pada bagian ”File name”. Ingat! Simpan berkas dalam format HTML. Pada bagian “Save as type”, silakan pilih “All Files (*.*)” dan Anda dapat memberinya nama beserta formatnya secara langsung - nama dan format yang dimaksud adalah “index.html”.
  5. Jika sudah, tekan tombol “Save” untuk memproses penyimpanan berkas.
  6. Luar biasa! Berkas tersebut sudah dapat kita lihat pada folder yang kita tentukan sebelumnya. Berikut adalah kondisi direktori WebDasar milik kami saat ini.

Mengidentifikasi Elemen pada Halaman Profil.

  1. Membuka hasil akhir pada latihan sebelumnya. Jika belum punya, silakan siapkan proyek berisi berkas index.html dengan kode berikut.
  2. Pada berkas index.html, silakan tambahkan struktur dasar dokumen HTML di posisi teratas. Kode yang perlu ditambahkan adalah kode bercetak tebal.
  3. Karena seluruh konten yang ditampilkan ada dalam elemen body, silakan masukkan seluruh konten artikel di elemen body.
  4. Selanjutnya, kita akan mengidentifikasi konten artikel ini. Carilah konten yang paling penting atau topik utama pembahasan dari artikel serta bungkus konten tersebut dengan elemen h1; lalu, konten yang merupakan pemaparan dari topik tersebut serta bungkus konten tersebut dengan elemen p; dan seterusnya.
  5. Kita belum selesai sampai di sini. Tidak hanya elemen body, untuk memperlengkap dokumen ini, kita akan melengkapi elemen dalam elemen head. Silakan tambahkan kode bercetak tebal berikut pada index.html.
  6. Seharusnya tab browser Anda akan menampilkan judul webpage dengan baik.

Menambahkan List pada Halaman Profil.

  1. Silakan buka proyek Halaman Profil terakhir dengan VS Code.
  2. Kita akan menambahkan daftar navigasi menggunakan elemen list. Silakan buka berkas index.html dan tambahkan elemen unordered list berikut di bawah dari elemen paragraf pertama pada berkas HTML.

Menambahkan Gambar pada Halaman Profil.

  1. Silakan buka proyek Halaman Profil terakhir dengan VSCode.
  2. arena kita akan menampilkan elemen-elemen gambar, silakan unduh aset-aset gambar yang digunakan pada halaman-profil-bandung-images.zip.
  3. Jika sudah, silakan pindahkan berkas hasil unduhan pada halaman proyek. Pastikan berkas diletakkan sejajar dengan berkas index.html.
  4. Lakukan ekstrasi berkas halaman-profil-bandung-images.zip.
  5. Sampai dengan langkah ini, berkas halaman-profil-bandung-images.zip sudah tidak digunakan dan bisa dihapus.
  6. Sebelum lanjut, pastikan isi folder assets → image ada empat buah berkas gambar.
  7. Pada folder WebDasar, kita sudah memiliki dua isi, yaitu folder bernama “halaman-profil-bandung-images” dan berkas index.html. Silakan ubah dulu nama folder “halaman-profil-bandung-images” menjadi “assets”. Tujuannya hanya untuk mempermudah pemanggilan folder saja ketika digunakan.
  8. Jika aset gambar sudah siap, mari kita terapkan pembelajaran elemen img pada proyek ini. Silakan buka berkas index.html dan tambahkan elemen-elemen gambar di bawah elemen heading sesuai kontennya masing-masing. Kode yang perlu ditambahkan memiliki cetakan tebal.

Menerapkan Elemen Anchor pada Halaman Profil.

  1. Membuka proyek Halaman Profil terbaru.
  2. Menambahkan kode berikut.
  3. Setelah menambahkan elemen anchor pada list item, tampilan navigation list akan menjadi hyperlink.
  4. Tambahkan ID pada tiap-tiap elemen h2.
  5. Setelah menambahkan ID tambahkan pada atribut href dengan ID yang terdapat di elemen h2.
  6. Maka tampilan akhirnya akan menjadi seperti ini.
  7. Seharusnya navigasi sudah berfungsi dengan baik. Anda bisa klik salah satu navigatin item tersebut untuk menuju topik yang sesuai keinginan anda.

Menerapkan Elemen Header, Main, dan Footer di Halaman Profil.

  1. Membuka proyek Halaman Profil.
  2. Kita akan membagi konten yang berada di halaman body dengan menambahkan elemen header terlebih dahulu.
  3. Bungkus elemen list yang berisikan navigasi dengan elemen nav.
  4. Kita akan masuk ke bagian konten kaki terlebih dahulu dengan menambahkan kode berikut.
  5. Langkah selanjutnya kita akan membungkus elemen diantara header dan elemen footer dengan elemen main.

Menerapkan Elemen Div, Aside, Article, dan Section di Halaman Profil.

  5. Bagus! Konten sudah terstruktur dengan lebih baik.

Menerapkan Elemen Tabel pada Halaman Profil.

  1. Silahkan buka proyek Halaman Profil terakhir dengan VS Code.
  2. Sebelum kita menambahkan konten ke HTML, mari kita atur kembali struktur elemen dalam elemen aside.
  3. Kita perlu mengelompokkan kembali elemen yang ada didalam elemen article karena akan menambahkan konten lain. Kelompokkan seluruh elemen yang ada pada elemen article saat ini sebagai header dan tambahkan elemen baru dibawah elemen header, yaitu section.
  4. Kemudian pada elemen section, kita masukkan konten berikut.
  5. Itulah konten yang ditampilkan pada elemen aside. Sesuaikan konten tersebut dengan menerapkan elemen yang tepat, salah satunya gunakan elemen tabel untuk menampung sebagian kontennya.
  6. Hasilnya akan lebih efisien dan rapi jika menggunakan elemen tabel.

Latihan Styling untuk Halaman Profil.

  1. Silahkan buka proyek Halaman Profil terakhir dengan VS Code.
  2. Untuk menerapkan styling, kita akan menerapkannya pada folder terpisah.
  3. Tambahkan folder baru didalam folder "asset image" kalian lalu tambahkan folder baru bernama "styles".
  4. Folder baru ini akan kita gunakan untuk menyimpan berkas CSS. Silahkan buka folder tersebut dan buat berkas baru dengan nama "style.css". Untuk membuat berkas styling, tentu anda perlu menggunakana format .css
  5. Folder styles dan berkas styles.css sudah siap untuk dimanfaatkan. Kita akan menggunakan berkas ini untuk styling proyek Halaman Profil.

Latihan Menulis Rules pada Berkas CSS

  1. Silahkan buka proyek Halaman Profil terakhir dengan VS Code.
  2. Untuk latihan awal, kita akan menuliskan rule styling pada beberapa elemen HTML yang tersedia dalam Halaman Profil. Kita akan mengubah tipe font. Tetapkan tipe font 'sans-serif' sebagai font yang digunakan.
  3. Untuk styling kedua, kita menetapkan juga warna pada setiap elemen h2 dan h3 dengan menuliskan kode berikut.
  4. Styling terakhir adalah memberikan warna background, warna teks, dan padding pada elemen footer. Silahkan tambahkan kode berikut.
  5. Akhir dari keseluruhan kode pada berkas style.css akan tampak seperti berikut.

Mwnghubungkan Berkas CSS dengan Halaman Profil.

  2. Setelah menuliskan rules pada berkas style.css, langkah selanjutnya adalah menghubungkan berkas style.css

Latihan Menggunakan Font dari Google Fonts

  1. Silahkan buka proyek Halaman Profil terakhir dengan VS Code.
  2. Untuk menggunakan font yang disediakan Google Fonts, silahkan kunjungi website Google Fonts pada fonts.google.com
  3. Saat ini, kita akan mencari font dengan nama "Quicksand", lakukan pencarian font tersebut pada search bar yang tersedia pada wensite.
  4. Jika sudah menemukan font yang dicari, tekan tombol card(kartu) dari font tersebut. Anda akan diarahkan ke halaman baru. Jika halaman digulir (scroll ke bawah), Anda akan melihat beberapa tingkatan style dari font ini.
  5. Kali ini, kita akan mengguanakan Quicksand dengan tingkat ketebalan 400 (reguler) dan 700 (bold). Untuk memilih font tersebut, silahkan tekan tombol "+" sehingga Google Fonts akan menyimpannya sebagai Selected Families-ini seperti fitur cart pada online shop.
  6. Setelah selesai memilih font yang dicari, perhatikan di bagian "Use on the web" pada sidebar bernama Selected Family. Pada bagian tersebut, kita diperlihatkan dua cara untuk menghubungkan font family yang dipilih pada halaman web dan cara menyetel font ini pada CSS rules.
  7. Kali ini, kita akan menggunakan cara link. Sehingga lakukan copy dengan cara blok seluruh kodenya dan klik kanan lalu copy atau klik tombol tumpukan kertas pada pojok kanan bawah kotak abu-abu. Jika sudah, silahkan lakukan paste dalam elemen head pada index.html. Berikut adalah kesulurhan dalam elemen head.
  8. Mantap! Sekarang, font Quicksand sudah terpasang pada dokumen HTML dan kita dapat memanfaatkannya pada CSS. Silahkan buka berkas style.css dan kita ubah nilai properti font-family pada body dengan menambahkan nilai 'Quicksand' sebelum generic font sans-serif.
  9. Dengan demikian, tampilan font pada halaman web akan menggunakan Quicksand.

Latihan Menetapkan Ukuran, Ketebalan, dan Aligment Teks di Halaman Profil

  1. Silahkan buka proyek Halaman Profil terakhir dengan VS Code
  2. Latihan pertama adalah mengatur ukuran font menjadi 18px pada navigation list(nav). Silahkan tambahkan kode bercetak tebal berikut pada style.css.
  3. Setelah ukuran, kita atur ketebalan menjadi 400 dan hilangkan underline (garis bawah) pada elemen tersebut sehingga isi block rulenya menjadi seperti berikut.
  4. Seharusnya sekarang ukuran teks pada daftar navigasi akan tampak lebih besar dan tidak memiliki garis bawah.
  5. Namun, ada yang kita bisa tingkatkan dari styling ini. Bagaimana jika kita mengubah style-nya jika elemen tersebut disorot dengan kursor? Kita bisa memberikan efek tebal pada teks ketika hal tersebut terjadi. Kita akan memanfaatkan pseudo-class :hover pada elemen a. Silahkan tambahkan kode bercetak tebal berikut.
  6. Nah, kita sudah selesai dengan daftar navigasi. Selanjutnya, kita akan membuat teks pada elemen header dalam elemen samping aside menjadi rata tengah. Kita kana memanfaatkan properti text-align. Namun, sebelum itu, untuk memudahkan proses styling, kita akan memberikan atribut class pada elemen article dengan nilai "profile".
  7. Setalah itu, kita dapat menentukan selector dengan mudah untuk mengatur header. Silahkan tambahkan kode berikut pada berkas style.css.
  8. Sampai langkah ini seharusnya, tampilan elemen hader pada konten samping akan tertata menjadi rata tengah.
  9. Terakhir kita buat juga teks pada footer menjadi rata tengah dan tebal dengan menambahkan properti text-align: center dan font-weight: bold pada footer selector. Berikut adalah keseluruhan isi dari berkas style.css.
